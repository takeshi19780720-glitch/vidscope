## 2. Types of Information We Collect

We use the **YouTube Data API v3** provided by Google LLC ("YouTube"). Through this API, the Service retrieves:

- Video metadata, titles, descriptions, view counts, and statistics via `videos.list`

- Channel information including subscriber counts via `channels.list`
- Video category listings via `videoCategories.list`

YouTube API data is retrieved solely to provide the Service's functionality and is subject to [Google's Privacy Policy](https://policies.google.com/privacy).

## 8. YouTube API Services Compliance

VidScope uses **YouTube API Services** (YouTube Data API v3) provided by Google LLC. Our use of these APIs is subject to and fully compliant with the [YouTube API Services Terms of Service](https://developers.google.com/youtube/terms/api-services-terms-of-service).

### 8.1 YouTube API Usage

The following YouTube API endpoints are used solely to provide core Service functionality:

- `search.list` — Retrieve video search results based on user-submitted keywords and date filters
- `videos.list` — Retrieve video details and statistics for display within the Service
- `channels.list` — Retrieve channel information and subscriber counts for display
- `videoCategories.list` — Retrieve video category listings for filtering and display

### 8.2 Handling of YouTube API Data

- YouTube API data is used exclusively to display information within the Service to the requesting user.
- We do not use YouTube API data to build user profiles, marketing lists, or for advertising purposes.
- We do not use YouTube API data to train, fine-tune, or improve any artificial intelligence or machine learning model.
- YouTube API data retrieved on the client side is held only in memory or temporary cache during an active session and is not persisted to our servers beyond what is necessary for session continuity.

### 8.3 Caching Strategy

To minimize unnecessary API calls and comply with YouTube API quota policies:

- Short-lived client-side caching is used within individual sessions.
- No YouTube API response data is stored in a persistent server-side database.
- Cached data is invalidated upon session termination or when the user logs out.

### 8.4 Compliance Statement

By using the VidScope Service, you acknowledge that your use of YouTube-sourced content is also subject to the [YouTube Terms of Service](https://www.youtube.com/t/terms) and [Google's Privacy Policy](https://policies.google.com/privacy). We are committed to maintaining full compliance with YouTube API Services Terms of Service requirements, including data deletion and user rights obligations.
