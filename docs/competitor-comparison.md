## 補足：比較項目の選定理由

| 比較項目 | 選定理由 |
| :--- | :--- |
| **主要用途** | このサービスが何のために作られたかを示す |
| **対象ユーザー** | 誰に最適かを分かりやすくする |
| **動画検索** | VidScope の中核機能。他社との差別化ポイント |
| **ランキング表示** | 人気動画・チャンネルを整理して見せる機能 |
| **推定収益分析** | VidScope の特徴的機能の一つ |
| **SEO 最適化支援** | 競合の強み分野。VidScope は今後強化候補 |
| **自チャンネル連携** | 自分の動画データを取り込むかどうか |
| **ブラウザ拡張** | YouTube サイト内で作業する際の利便性 |
| **モバイル対応** | スマホから使いやすさ |
| **多言語対応** | VidScope が強みとするポイント |
| **料金体系** | ユーザーが最も気にする選択材料 |
| **セットアップの手間** | 手軽に始められるかどうか |
| **広告/アフィリエイト** | UI の煩わしさや信頼性に関わる |
| **API コンプライアンス** | VidScope が現在対応中の重要項目 |

## 匿名化された比較先について

社内資料としての参照性を保つため、実名は掲載せず A社〜D社として匿名化しています。実際の対応関係は社内限りで別途管理してください。
