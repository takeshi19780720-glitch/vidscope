Features

検索を、もっと自由に。

VidScopeは、YouTubeの標準検索では届かない高精度なフィルタリングを実現します。

🌍

多言語・地域フィルター

日本語、英語、韓国語、中国語など言語別に動画を絞り込み。地域指定でローカルトレンドも発見。

📊

再生回数ランキング

再生回数順の正確なソートで、人気動画を一目で把握。エンゲージメント率も自動計算。

⏱️

再生時間フィルター

ショート動画、通常動画、長尺動画を簡単に切り替え。目的に合った動画だけを表示。

💰

推定収益シミュレーション

CPMベースで動画の推定収益を自動算出。ジャンル別CPMのカスタム設定にも対応。

🔑

マルチAPIキー

複数のAPIキーを登録可能。クォータ超過時に自動ローテーションで、検索が止まらない。

🔒

セキュア設計

管理者認証、レート制限、セキュリティヘッダーを標準装備。安心して運用できます。

FAQ

よくある質問

APIキーは自分で用意する必要がありますか？
はい、YouTube Data API v3のAPIキーが必要です。Google Cloud Consoleから無料で取得できます。設定画面に取得手順のガイドも用意しています。
APIのクォータ制限はどうなりますか？
YouTube Data APIのデフォルトクォータは1日10,000ユニットです。VidScopeは複数APIキーの自動ローテーションに対応しているため、クォータを効率的に活用できます。
無料プランに期限はありますか？
いいえ、無料プランに期限はありません。検索回数の制限はありますが、基本機能をずっとお使いいただけます。
データは安全ですか？
VidScopeはユーザーデータを一切収集・保存しません。検索クエリはYouTube APIに直接送信され、サーバーにログとして残ることもありません。
プランの変更やキャンセルはできますか？
はい、いつでもプランの変更・キャンセルが可能です。日割り計算で返金対応いたします。
